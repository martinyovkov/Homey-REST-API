# Homey-REST-API

## Running

1. Create a .env file in the server.js directory
The required values are: DB_QUERYSTRING, COOKIE_SESSION_NAME, SECRET, ENVIRONMENT, PORT, ORIGIN
An example file will be applied

2. run npm install
3. run npm start
